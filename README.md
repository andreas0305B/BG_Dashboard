🎲 Backgammon Championship Dashboard
This is a Streamlit web app to explore DailyGammon championship data.
👉 Try it live on Streamlit
✨ Features
League Table with scores
Score Matrix: results between players (clickable links to the games)
Match ID Matrix: all match IDs (clickable links to DailyGammon)
Season & Group selectors for quick navigation
All tables are styled for readability and compact display.
